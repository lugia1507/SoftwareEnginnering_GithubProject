// 01 — Project Scope

Personal Portfolio Websites

Each member of Group-6 built a personal portfolio website introducing themselves — who they are, the course they are studying, their skills, and a sample project. Every portfolio lives in its own GitHub repository and was built using the core GitHub collaboration workflow.

Commit

Every change tracked with clear, meaningful commit messages across all three repositories.

Branch

Feature branches keep experimental work isolated before merging into the main branch.

README

Each repository documents its purpose, structure, and setup in a well-written README.

Issue

Tasks, bugs, and ideas logged as issues to plan and track our progress transparently.

Pull Request

Changes reviewed through pull requests before merging — the professional workflow.

Built With

HTML for structure, Tailwind CSS for responsive design, and JavaScript for interactivity — hosted and versioned on GitHub.
