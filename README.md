# HarThaPel — Group 5 Portfolio Hub

Welcome to the central repository for **Group 5** from **MIBA College / UNIMY (Computer Science, Academic Year 2025 – 2028)**. This landing hub acts as our shared project space, showcasing individual personal portfolios and demonstrating our mastery of collaborative Git workflows.

> **> Three students. Three portfolios. One team. ▊**

---

## 🚀 Project Overview

The goal of this project was to establish a collaborative, professional development environment. Each member built a personal portfolio website introducing themselves, their skills, their academic journey, and sample projects. Every portfolio lives in its own repository and is linked through this hub.

### Built With
*   **HTML5** – Semantic structural foundation.
*   **Tailwind CSS** – Utility-first frameworks for custom theme styling and responsive layouts.
*   **JavaScript (ES6+)** – Interactive features including custom starfields, 3D card tilt effects, and scroll reveals.
*   **Git & GitHub** – Full version control and workflow automation.

## 🛠️ GitHub Collaboration Workflow

To replicate an industry-standard development cycle, our team structuralized progress using the core GitHub features:

*   **Issues:** Tracked bugs, conceptual features, and academic requirements transparently.
*   **Branching:** Features were developed in isolated branches to prevent disruptions to the `main` branch.
*   **Commits:** Maintained clear, concise, and descriptive commit logs across all repositories.
*   **Pull Requests (PRs):** Peer-reviewed code variations before merging to ensure code quality and deployment safety.
