## 🚀 Project Description

The goal of this project was to establish a collaborative, professional development environment[cite: 1]. Each member built a personal portfolio website introducing themselves, their skills, their academic journey, and sample projects[cite: 1]. Every portfolio lives in its own repository and is linked through this hub[cite: 1].

---

## ⚡ Features

*   **Interactive Starfield Background:** A custom HTML5 Canvas particle system that dynamically handles window resizing[cite: 1, 2].
*   **Accessibility Control:** Animation scripts automatically detect and respect user system preferences (`prefers-reduced-motion`)[cite: 2, 3].
*   **Scroll-Reveal API:** Seamless fade-in movements powered by the `IntersectionObserver` API[cite: 2, 3].
*   **3D Hover Tilt Dynamics:** An interactive perspective tilt applied to team profile cards on mouse movement[cite: 2, 3].
*   **Built With:** Developed natively using HTML5, Tailwind CSS, JavaScript (ES6+), and Git/GitHub[cite: 1].

## 🛠️ How to Run / Open the Project

Since this project is built entirely using client-side frontend technologies, it runs directly in the browser without any build steps or server configurations.

### Option 1: Open Locally
1. **Clone or Download** this repository to your local machine.
2. Navigate into the root project directory.
3. Double-click the `index.html` file to instantly launch it in your default web browser[cite: 1].

### Option 2: Live Deployment
Simply host the root folder via **GitHub Pages** (or any static hosting provider) to view the project live on the web.
